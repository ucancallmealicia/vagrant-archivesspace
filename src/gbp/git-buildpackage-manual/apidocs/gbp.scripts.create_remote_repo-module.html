print_config(remote, branches)

 

Print out the git config to push to the newly created repo.

>>> print_config({'name': 'name', 'url': 'url'}, ['foo', 'bar'])

        url = url
        fetch = +refs/heads/*:refs/remotes/name/*
        push = foo
        push = bar
[branch "foo"]
        remote = name
        merge = refs/heads/foo
[branch "bar"]

        merge = refs/heads/bar
